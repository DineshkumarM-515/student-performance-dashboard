# Student Performance Dashboard

This project is an interactive Tableau dashboard created to analyze student academic performance using data visualization techniques.

## Features
- Department-wise performance analysis
- Attendance analysis
- Semester-wise performance trends
- Interactive filters and visual insights

## Tools Used
- Tableau Public
- Microsoft Excel
